cooling rate vs temperature for the five cases grid_*coronal*.in in tsuite / auto

These have been published in
http://adsabs.harvard.edu/abs/2013MNRAS.429.3133L 
http://adsabs.harvard.edu/abs/2014MNRAS.440.3100W

The solar/solar full H2 plots compare the default cooling with that done
with large models turned on.  These differ only around 5000 K.

The primal / primal H2 has the default universal fit to H2 cooling and
the large molecule.
